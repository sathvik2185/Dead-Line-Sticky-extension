# Dead-Line-Sticky-extension
A lightweight Chrome extension that helps students stay on track by displaying persistent sticky reminders for important deadlines. Designed to show non-intrusive pop-up alerts at the corner of the browser, ensuring you never miss a submission while browsing.An intuitive deadline reminder extension for students that delivers timely sticky notifications inside Chrome, improving productivity and reducing missed academic deadlines.

✅ Advantages
• Prevents missed deadlines by displaying persistent, visible reminders directly inside the Chrome browser.
• Non-intrusive sticky pop-ups appear at a fixed corner of the window, ensuring reminders without interrupting browsing.
• Boosts student productivity by keeping academic tasks and due dates constantly in view.
• Simple and lightweight design, ensuring fast performance with minimal browser resource usage.
• Chrome-exclusive integration allows seamless functionality using Chrome Extension APIs.
• User-controlled reminders, enabling students to add, update, and manage deadlines easily.
